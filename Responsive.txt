Kỹ thuật thiết kế trang Web có Responsive
1. Dùng CSS thay đổi kích thước phù hợp cho các thành phần hiển thị trên website (hình ảnh, cỡ chữ, nút bấm,...)
2. Dùng CSS để ẩn/ hiện các thành phần phù hợp theo kích thước màn hình

Cách ẩn hiện ví dụ
display: none; ( để ẩn )
display: block; ( để hiện trở lại )
